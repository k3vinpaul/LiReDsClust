## Raw Files

### `LRD_MarIRAC.fits`

Primary galaxy catalog. 3 341 rows, 13 columns:

| Column | Type | Description |
|---|---|---|
| `ID` | int | Source identifier |
| `RA` | float64 | Right ascension [degrees] |
| `DEC` | float64 | Declination [degrees] |
| `z` | float64 | Photometric redshift |
| `z_err` | float64 | Redshift uncertainty |
| `MUV` | float64 | UV absolute magnitude |
| `MUV_err` | float64 | MUV uncertainty |
| `beta_UV` | float64 | UV spectral slope |
| `beta_UV_err` | float64 | Œ≤_UV uncertainty |
| `beta_opt` | float64 | Optical spectral slope |
| `beta_opt_err` | float64 | Œ≤_opt uncertainty |
| `IRAC-detected` | bool | True if source detected by Spitzer/IRAC |
| `IRAC-footprint` | bool | True if source lies within IRAC footprint |

**IRAC flag counts**:
- `IRAC-footprint = True`: 1 509 galaxies
- `IRAC-detected  = True`: 29 galaxies

### `RA_sim_alpha0.5.npy` / `DEC_sim_alpha0.5.npy`

N-body simulation coordinate arrays provided by collaborators.
Used to generate simulation-based error bars via random sky rotations.

> **Known column swap**: despite their file names, `RA_sim_alpha0.5.npy`
> contains **Dec** values and `DEC_sim_alpha0.5.npy` contains **RA** values.
> This swap is corrected in `src/errors.py` and `scripts/run_simulations.py`.

### `masks/mask_map_healsparse_EDFS_v1.fits`

HealSparse survey mask for the Euclid Deep Field South (EDFS), version 1.
Used to:
1. Remove galaxies outside the observed footprint (`src.catalog.apply_sky_mask`).
2. Generate masked random catalogs (`src.randoms.generate_random_catalog`).
3. Prepare the OneCovariance mask input (downgraded to nside=1024).

### `simulations/`

N-body simulation sky catalogs provided by collaborators. Used to generate
simulation-based error bars (see `scripts/run_simulations.py`).

**Column-swap bug**: Despite their filenames, `RA_sim_*.npy` stores **Dec** values
and `DEC_sim_*.npy` stores **RA** values. This swap is corrected automatically.

| File pair | Config key | Œ± value |
|---|---|---|
| `RA/DEC_sim_alpha0.5.npy` | `alpha0.5` | 0.5 |
| `RA/DEC_sim_alpha1.npy` | `alpha1` | 1 |
| `RA/DEC_sim_alpha3.npy` | `alpha3` | 3 |
| `RA/DEC_sim_noalpha.npy` | `default` | unlabelled |

Select which simulation to use via `simulations.active` in `config/analysis.yml`
or with `--sim-name` on the command line.

### `masks/mask_healpix_nside1024.fits`

Standard HEALPix map derived from the HealSparse mask by downgrading to
nside=1024 (pixel area ‚âà 12 arcmin¬≤). Required by OneCovariance.
Generated by `src.covariance.convert_healsparse_to_healpix` if missing.

## Processed Files

Random catalogs are cached to avoid regeneration on every run. They are
gitignored (large binary files). Regenerate with:

```bash
python scripts/run_2pacf.py --sample <sample_name> --regen-randoms --no-plots
```

Available sample names: `irac_footprint_masked`, `irac_footprint_unmasked`,
`irac_detected_masked`, `irac_detected_unmasked`, `no_irac_footprint_masked`.
